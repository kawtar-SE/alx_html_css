# alx_html_css
 **Headphones Website** 

Welcome to the **Headphones Web Page** project — a sleek, responsive landing page built with pure **HTML** and **CSS**, following clean design principles and modern interaction effects.


---

## Project Overview

This project  demonstrates the structure of a basic webpage with several sections:
**Header and Navigation:** Includes links to "what we do", "our results", and "contact us".
**Contact Section:** Provides a contact form where users can enter their name, email, and message to get in touch.
**Footer:** Displays the brand logo, social media links, and copyright information, ensuring a clean and consistent design.

---

## Features
**Immersive Audio Experience:** Discover the latest in sound design and high-quality headphones that deliver crystal-clear audio.
**Service Highlights:** Learn what makes the brand unique in the “What We Do” section featuring icons and short descriptions.
**Contact Form Integration:** Easily reach out using a clean, user-friendly contact form.
**Responsive Design:** The page automatically adapts to all screen sizes — desktop, tablet, and mobile.

---

## Interactions & Behavior

the web page must switch to the mobile version when the screen width is 480px or less
links hover/active: #FF6565 / button hover/active: opacity: 0.9 / max width of the content: 1000px centered in the page
