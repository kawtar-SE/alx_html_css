# CSS Advanced Project - alx_html_css

## Objectives
- Practice advanced HTML and CSS concepts.
- Implement Flexbox layouts.
- Make pages responsive for mobile devices.
- Customize styles including colors, fonts, logos, borders, and backgrounds.
- Follow a design provided in Figma.

## Figma Design
- [Page in Figma](https://www.figma.com/design/dyYL6Ku4WG7vsdpwvlcJZC/Homepage?node-id=0-1&t=5ke1rnFbpIq31lFi-1)
- Fonts used: Source Sans Pro, Spin Cycle OT
- Images for the project: [Download images](https://intranet-projects-files.s3.amazonaws.com/webstack/Homepage.fig)

## Project Structure
- `index.html` - main HTML page
- `base.css` - base CSS rules
- `styles.css` - custom styles for layout and design
- `README.md` - project description

## Notes
- Some measurements in Figma are floats – you may round them.
- Use Unicode characters for logos or images as needed.
- Keep Flexbox layout strategy intact.
